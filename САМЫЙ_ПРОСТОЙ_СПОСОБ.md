# Самый простой способ: Выгрузить код в GitHub через Cursor

## Способ 1: Через панель Source Control (самый простой, без терминала)

### Шаг 1: Открой панель Source Control

1. В Cursor слева найди иконку **ветки** (или нажми `Cmd+Shift+G`)
2. Откроется панель **Source Control**

### Шаг 2: Инициализируй Git (если еще не сделано)

1. Если видишь кнопку **"Initialize Repository"** — нажми её
2. Если видишь список файлов — Git уже работает, переходи к шагу 3

### Шаг 3: Подключи к GitHub

1. Нажми **"..."** (три точки) вверху панели Source Control
2. Выбери **"Remote"** → **"Add Remote"**
3. В поле **Name** введи: `origin`
4. В поле **URL** вставь ссылку на свой репозиторий:
   - Открой GitHub → свой репозиторий → кнопка **"Code"** (зеленая)
   - Скопируй ссылку (HTTPS)
   - Вставь в поле URL
5. Нажми **"Add"**

### Шаг 4: Отправь код в GitHub

1. В панели Source Control нажми **"+"** рядом с файлами (чтобы добавить все в коммит)
2. Вверху в поле **Message** введи: `Initial commit`
3. Нажми кнопку **"Commit"** (галочка справа)
4. Нажми **"..."** (три точки) → **"Push"**
5. Выбери `origin` и ветку `main`
6. Нажми **"Push"**

**Готово!** Код отправлен в GitHub.

---

## Способ 2: Через веб-интерфейс GitHub (если способ 1 не работает)

Если панель Source Control не работает, можно загрузить файлы прямо через сайт GitHub:

1. Открой свой репозиторий на GitHub
2. Нажми **"Add file"** → **"Upload files"**
3. Перетащи все файлы из папки `1_дата-аналитика` (кроме `.env`)
4. Нажми **"Commit changes"**

---

## Как обновлять код в GitHub (когда что-то меняется)

### Через панель Source Control:

1. Открой панель Source Control (`Cmd+Shift+G`)
2. Увидишь список измененных файлов
3. Нажми **"+"** рядом с файлами
4. Введи сообщение (например, "Обновлен код")
5. Нажми **"Commit"**
6. Нажми **"..."** → **"Push"**

**Всё!** Изменения отправлены в GitHub.

---

## Что делать, если не видишь панель Source Control

1. В меню Cursor: **View** → **Source Control**
2. Или нажми `Cmd+Shift+G`

---

## Важно

- **Файл `.env` не должен попасть в GitHub** — он в `.gitignore`, Cursor автоматически его пропустит
- **Все остальные файлы** будут отправлены
- **После каждого Push** GitHub Actions автоматически запустится (если настроен workflow)

---

## Если что-то не работает

**Не вижу панель Source Control:**
- View → Source Control (или `Cmd+Shift+G`)

**Не могу подключить репозиторий:**
- Проверь ссылку на GitHub (должна начинаться с `https://github.com/`)

**Не могу запушить:**
- Возможно, нужна авторизация (GitHub попросит логин/пароль или токен)
